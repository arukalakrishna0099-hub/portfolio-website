# 🚀 SkillVerse - Personal Portfolio Website

A modern, responsive, and interactive portfolio website designed to showcase professional skills, experience, projects, and services. Built using HTML5, CSS3, and JavaScript with a clean UI, smooth animations, and mobile-friendly design.

## 🌟 Features

- Responsive Design
- Interactive Resume Section
- Dynamic Project Carousel
- Animated Hero Section
- Services Showcase
- Skills & Experience Timeline
- Contact Form
- Social Media Integration
- Modern Dark-Themed UI
- Mobile-Friendly Layout

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/arukalakrishna0099-hub/portfolio-website.git
```

Navigate to the project folder:

```bash
cd portfolio-website
```

Run locally:

```bash
Open index.html in your browser
```

or use VS Code Live Server.
